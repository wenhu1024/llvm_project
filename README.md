
## Environment 
   1. Ubuntu 22.04
   2. Cmake 3.22
   3. LLVM toolchain version 14.0
<br/>


## how to run 
cd [this_project_name]

```bash
mkdir build && cd build && cmake .. && cmake --build . && ./kaleidoscope

```

<br/>

## NOTE
   1. after you can input something in terminal
   2. input ctrl+C or ctrl+D to end program
